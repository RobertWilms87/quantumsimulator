Quantum Circuit Simulator

Drag gates onto qubit lines, select a gate and add any number of controls, build arbitrary rotation gates, group subcircuits into reusable custom gates, and load standard circuits such as Grover, QFT, and a Shor period-finding circuit. Exact state-vector simulation is exponential, so large circuits are possible to draw, but simulation becomes expensive as the number of qubits grows.

Standard programs

To make a controlled version of any gate, including a grouped gate: click the gate, press Add/remove controls, then click control cells in the same column. Multiple controls are supported. Grouped gates are placed as one compact block and expanded only during simulation. Use Classical condition after a measurement for feed-forward; the |0⟩ tool performs a real reset.

Rotations accept expressions such as pi/2, -3*pi/8, or decimals. Use Import / Export to preserve the circuit together with all grouped-gate definitions. Qiskit and PennyLane Python imports parse supported numeric operations without executing the source file. Stim, QEC rounds, CSS decoders, exact analysis, hardware routing, optimization, and workspace tabs are available in Advanced lab.
